# Лабораторные работы №1-4

## Базы данных


### Описание

В данном репозитории представлены решения лабораторных работ по курсу "Базы данных". Работы выполнены на языке SQL с использованием учебной базы данных университета.

### Содержание

- **Лабораторная работа №1** — Запросы с соединением таблиц (JOIN)
- **Лабораторная работа №2** — Агрегатные функции и группировки
- **Лабораторная работа №3** — Подзапросы и вложенные запросы
- **Лабораторная работа №4** — Оптимизация запросов с использованием CTE

### Использованные технологии

- PostgreSQL
- Подзапросы (IN, скалярные)
- Обобщённые табличные выражения (CTE)
- Агрегатные функции (COUNT, AVG, MIN)
- Регулярные выражения для фильтрации
- RIGHT JOIN, INNER JOIN
